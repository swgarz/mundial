🚀 Crea tu quiniela

Configúrala una sola vez. Obtendrás un enlace para compartir con quien quieras que participe.

¿No tienes la URL? Mira cómo crearla gratis (3 min, una sola vez)
  1. Entra a console.firebase.google.com con tu cuenta de Google.
  2. Clic en "Agregar proyecto". Ponle un nombre (ej. quiniela-attrapi), puedes saltar Google Analytics, y Crear proyecto.
  3. En el menú izquierdo: Compilación → Realtime Database → Crear base de datos. Elige la ubicación y selecciona "Iniciar en modo de prueba" → Habilitar.
  4. Entra a la pestaña "Reglas", borra lo que haya y pega esto, luego Publicar:
    { "rules": { ".read": true, ".write": true } }
  5. Arriba verás la dirección de tu base (algo como https://quiniela-attrapi-default-rtdb.firebasedatabase.app). Cópiala y pégala en el campo de arriba. ¡Listo!
